# 🎾 ピックルボール完全ガイド

ピックルボールのルール・スコアコール・コート寸法を完全網羅したWebアプリ。

## 機能

- 📋 基本ルール
- 📚 詳細ルール (サーブ / キッチン / スコア / ラリー)
- 📐 コート寸法ガイド
- 🎯 スコアコール練習シミュレータ
- 🎮 クイズモード
- 🎲 乱数表(ダブルスペア決め)
- 📅 ルール改定履歴
- 📖 ルール辞典 (59項目)

## 技術スタック

- React 18
- Vite
- Vanilla CSS-in-JS

## ローカルで動かす

```bash
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開く。

## ビルド

```bash
npm run build
```
